Scratch Quest Game

Quest Game

This is my quest game. It is my final CS First project for Introduction to Programming and Robotics, and it involves a character set out to find gold. You can control the character using the arrow keys, and must journey throughout the land to find the gold. I learned how to broadcast messages within the code while i was making this project. For some reason, the house, the cave, and the forest did not reappear when the character returned to the town, even after the code received messages telling it to make them appear. I fixed this by making them appear when the backdrop number equals their number. Add-Ons in this game are Peek-A-Boo and Obstacle Obstruction. This means the gold will only appear on the screen when close to it, and you wont be able to walk on trees, tables, logs, etc. Play by pressing the green flag...

CMU Python Group Picture

CMU Collaborative Task

This is a picture some friends and I made using python. The picture was a collaborative task in the CMU Python course that we were doing in AP Computer Science Principles. We coded shapes and labels in different sizes and colors to create this picture. The setting is 5 adolescent individuals camping outside of McDonalds in San Francisco to be the first ones to receive the new Travy Patty, but they are doing their favorite activites so they do not get bored. I designed the McDonalds building in the background, and the ping pong activity going on in the bottom right.
